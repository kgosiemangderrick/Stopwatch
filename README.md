Stopwatch

This is a stopwatch, build to help Athlets time their progress. Can record laps,start,stop and reset. Anyone can use the stopwatch, very intresting and challenging. Built with HTML, CSS, and JavaScript.

Live Demo
https://kgosiemangderrick.github.io/Stopwatch/

Features
- Responsive design that works on all devices
- Smooth scrolling navigation
 

Technologies Used
- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
